# Submission Instructions

## How to Submit

1. Copy this entire `/starter` folder into a **new GitHub repo**
2. Rename the repo something like `curator-cards-[yourname]`
3. Complete all files:
   - `CURATOR_NOTES.md` — your research thinking
   - `AGENTS.md` — your AI instructions (with filled vocabulary table)
   - `PROMPT_LOG.md` — at least 3 logged prompts
   - `CLAIMS.md` — at least 1 verified claim per card
   - `index.html` — your 3 Curator Cards
4. Enable GitHub Pages (Settings → Pages → Source: Deploy from branch → `main`, folder: `/ (root)`)
5. Make at least **5 meaningful commits** with descriptive messages
   - Bad: `update`, `fix`, `final`, `done`
   - Good: `add bauhaus card first draft`, `revise swiss style after expert critique`, `verify claims and add sources`
6. Submit your **repo link** in Canvas

## Checklist Before Submitting

- [ ] Site is live on GitHub Pages
- [ ] 3 cards, each 80–120 words
- [ ] 5 vocabulary tags per card (domain terms, not vibes)
- [ ] 2 working example links per card
- [ ] "Not to be confused with…" on each card
- [ ] Accountability note on each card
- [ ] `CURATOR_NOTES.md` filled in
- [ ] `AGENTS.md` has 15+ vocabulary terms defined
- [ ] `PROMPT_LOG.md` has 3+ logged prompts
- [ ] `CLAIMS.md` has 1+ verified claim per card + reflection
- [ ] Images (if any) have `alt` text
- [ ] 5+ meaningful git commits
