# Completed Sprints

Store finished sprint files here.

## Receipt Requirements (per sprint)
- Sprint file (`sprint00X.md`) with all checkboxes finalized
- Deploy/staging URL
- Screenshot(s)
- Verification notes (pass/fail)
- Commit hash or PR link

## Naming Pattern
- `sprint001.md`, `sprint002.md`, etc.
- Optional evidence folder: `sprint001_receipts/`
